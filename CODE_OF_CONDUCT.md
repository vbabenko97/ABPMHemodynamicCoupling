# Code of Conduct

## Our Standard

We want this repository to be a respectful, constructive place for scientific and engineering collaboration. Contributors should communicate clearly, critique ideas rather than people, and make room for different levels of experience.

## Expected Behavior

- Use welcoming and professional language.
- Share evidence, references, and reproducible examples when discussing scientific or technical disagreements.
- Accept good-faith review comments and respond with concrete follow-up.

## Unacceptable Behavior

- Harassment, personal attacks, or discriminatory language.
- Dismissing contributors without engaging the substance of their work.
- Publishing private or identifying information without permission.

## Reporting

Please contact the repository maintainers through a private GitHub channel if you need to report a conduct issue.
